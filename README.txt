=== HƯỚNG DẪN CHẠY DỰ ÁN NHẬN DIỆN CHỮ SỐ (MNIST) ===

1. CÀI ĐẶT MÔI TRƯỜNG:
   - Cần cài Python (3.10 trở lên).
   - Mở Terminal tại thư mục này và chạy lệnh:
     pip install -r requirements.txt

2. CÁCH CHẠY APP (LOCAL):
   - Chạy lệnh: python app.py
   - Mở trình duyệt truy cập: http://127.0.0.1:7860

3. LƯU Ý:
   - File model 'best_mnist_model.pth' phải nằm cùng chỗ với app.py
   - Nếu lỗi pip, thử dùng: py -m pip install -r requirements.txt
